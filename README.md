# Diwali-Sales-Data-Analysis-Using-Python-

This project analyses Diwali sales data to uncover customer purchasing behaviour, revenue trends, and high-performing product categories. The objective is to generate actionable business insights that can help optimise marketing strategies.

>> Objectives
Perform data cleaning and preprocessing for accurate analysis.
Conduct Exploratory Data Analysis (EDA) to identify sales patterns.
Analyse customer demographics (age, gender, occupation, state)
Identify top revenue-generating products and customer segments

>>Tools & Technologies
Python
Pandas – Data manipulation
NumPy – Numerical operations
Matplotlib & Seaborn – Data visualization
Google Colab

>>Key Insights
Identified high-value customer segments contributing maximum revenue(Women).
Analysed purchasing trends across different states and occupations.
Discovered top-performing product categories during Diwali season.
